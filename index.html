Project description

Gazetteer is a dynamic web application designed to satisfy the wanderlust of geography enthusiasts and curious minds alike. With Gazetteer, users embark on a virtual journey across the globe, uncovering a treasure trove of geographical information, weather updates, local news, and fascinating tidbits from Wikipedia, all at their fingertips.

For this project, I utilized a combination of HTML and CSS for structuring and styling web pages, JavaScript for client-side scripting, and PHP for server-side scripting.

To enhance Gazetteer's functionality and user interface, I incorporated various tools and libraries. Bootstrap was instrumental in ensuring a responsive and visually appealing design. Leaflet.js provided mobile-friendly interactive maps. jQuery simplified DOM manipulation, event handling, and asynchronous HTTP requests.
